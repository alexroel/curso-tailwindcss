## Introducción

Tailwind CSS es un framework CSS que permite crear interfaces de usuario de manera rápida y eficiente. A diferencia de otros frameworks como Bootstrap o Foundation, Tailwind CSS no proporciona componentes predefinidos, sino que se basa en clases utilitarias que permiten construir diseños personalizados.

---
## ¿Qué es Tailwind CSS?

Tailwind CSS es un framework CSS de bajo nivel que permite crear interfaces de usuario personalizadas. A diferencia de otros frameworks como Bootstrap o Foundation, Tailwind CSS no proporciona componentes predefinidos, sino que se basa en clases utilitarias que permiten construir diseños personalizados.

Tailwind CSS es altamente configurable y permite a los desarrolladores crear sus propios estilos y componentes. Esto lo convierte en una herramienta muy poderosa para el desarrollo de interfaces de usuario.

---
## Diferencias entre CSS y Tailwind CSS

La principal diferencia entre CSS y Tailwind CSS es que Tailwind CSS se basa en clases utilitarias, mientras que CSS se base en escribir estilos personalizados. Esto significa que en lugar de escribir CSS personalizado para cada elemento, puedes usar clases utilitarias predefinidas para aplicar estilos a los elementos.

De esta manera, Tailwind CSS permite crear diseños personalizados de manera más rápida y eficiente. Además, Tailwind CSS es altamente configurable y permite a los desarrolladores crear sus propios estilos y componentes.

Ejemplo de CSS:
```css
body {
  padding: 40px;
}

.btn {
    background-color: rgb(97, 123, 255); /* oklch(62.3% .214 259.815) */
    color: white;
    padding: 8px 16px;
    border-radius: 0.25rem;
    cursor: pointer;
    font-size: 18px;
  }
  
  .btn:hover {
    background-color: rgb(85, 110, 255); /* oklch(48.8% .243 264.376) */
  }
```

Ejemplo de Tailwind CSS:
```html
<!-- Botón con clases utilitarias de Tailwind CSS -->
<button class="bg-blue-500 text-white py-2 px-4 rounded text-lg cursor-pointer hover:bg-blue-300">Botón</button>
```

Como se puede ver, en el ejemplo de Tailwind CSS se utilizan clases utilitarias para aplicar estilos al botón, lo que permite crear diseños personalizados de manera más rápida y eficiente.

## Uso de Playground

Para empezar a trabajar con Tailwind CSS, puedes usar el [Playground de Tailwind CSS](https://play.tailwindcss.com/). Este es un entorno en línea que te permite experimentar con Tailwind CSS sin necesidad de instalar nada en tu computadora.

Otra forma de usar Tailwind CSS es a través de un CDN (Content Delivery Network). Esto te permite incluir Tailwind CSS en tu proyecto sin necesidad de instalarlo localmente. Para hacer esto, solo necesitas agregar el siguiente enlace en la sección `<head>` de tu archivo HTML:

```html
<link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css" rel="stylesheet">
```

Esto cargará la última versión de Tailwind CSS desde el CDN y podrás usarlo en tu proyecto sin necesidad de instalarlo localmente.

## Entorno de desarrollo

Para empezar a trabajar con Tailwind CSS, es necesario tener un entorno de desarrollo configurado. Esto incluye tener instalado Node.js y npm (Node Package Manager). Node.js es un entorno de ejecución de JavaScript que permite ejecutar código JavaScript en el servidor, mientras que npm es un gestor de paquetes que permite instalar y gestionar dependencias en proyectos de JavaScript.

Tambine un editor de código como Visual Studio Code, que es un editor de código gratuito y muy popular entre los desarrolladores. Visual Studio Code tiene una gran cantidad de extensiones y características que lo hacen ideal para trabajar con Tailwind CSS.

- [Node.js](https://nodejs.org/en/download/)
- [Visual Studio Code](https://code.visualstudio.com/)
- [Extensiones recomendadas para Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
    - [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode): Formateador de código que permite mantener un estilo de código consistente en todo el proyecto.

## Instalación de Tailwind CLI

Para instalar Tailwind CSS, primero debes crear un nuevo proyecto de Node.js. Puedes hacer esto ejecutando el siguiente comando en la terminal:

```bash
mkdir mi-portafolio
cd mi-portafolio
npm init -y
```

Esto creará un nuevo directorio llamado `mi-portafolio` y un archivo `package.json` dentro de él.

Luego deberás instalar Tailwind CSS y sus dependencias. Puedes hacer esto ejecutando el siguiente comando en la terminal:

```bash
npm install tailwindcss @tailwindcss/cli
```

Esto instalará Tailwind CSS y su CLI (interfaz de línea de comandos) en tu proyecto.

Si estamos usando el CLI, debemos crear un archivo CSS y agregar las directivas de Tailwind CSS. Puedes hacer esto creando un archivo llamado `src/input.css` y agregando lo siguiente:

```css
@import "tailwindcss";
```

Luego ejecutar el siguiente comando en la terminal para compilar el archivo CSS:

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
```
Esto compilará el archivo `src/input.css` y generará un nuevo archivo llamado `src/output.css` que contendrá todas las clases de Tailwind CSS.

Luego, debes crear un archivo HTML y agregar el enlace al archivo CSS en la sección `<head>`. El contenido de este archivo será el siguiente:

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi primer website con Tailwind CSS</title>
    <link rel="stylesheet" href="src/output.css">
</head>
<body>
    <h1 class="text-3xl font-bold text-center mt-10">Hola, mundo!</h1>
    <p class="text-center mt-5">Este es mi primer website con Tailwind CSS.</p>
</body>
</html>
```

Si abrimos el archivo `index.html` en el navegador, deberíamos ver un mensaje de bienvenida con un título y un párrafo centrados en la página.

## Configuración de proceso de compilación

Para no estar ejecutando el comando de compilación cada vez que hagamos un cambio en el archivo CSS, podemos configurar un proceso de compilación automático. Esto se puede hacer usando una herramienta como `npm scripts` o `Gulp`.

Para esto en el archivo `package.json` debemos agregar el siguiente script:

```json
"scripts": {
    "dev": "npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch"
}
```

Esto nos permitirá ejecutar el comando de compilación automáticamente cada vez que hagamos un cambio en el archivo CSS. Para ejecutar el script, solo debemos ejecutar el siguiente comando en la terminal:

```bash
npm run dev
```

Esto ejecutará el comando de compilación y generará el archivo `src/output.css` cada vez que hagamos un cambio en el archivo `src/input.css`.
