## Contenedor

Ahora aprenderemos a utilizar el contenedor de Tailwind CSS. El contenedor es una clase que se utiliza para centrar el contenido de una página web y establecer un ancho máximo para el contenido.
El contenedor se utiliza comúnmente en la mayoría de los proyectos de Tailwind CSS y es una parte fundamental del diseño de la página.

Para utilizar el contenedor, simplemente agrega la clase `container` a un elemento HTML. Por ejemplo:
```html
<div class="container mx-auto">
  <h1 class="text-3xl font-bold">Bienvenido a mi portafolio</h1>
  <p class="mt-4">Soy un desarrollador web y este es mi portafolio.</p>
</div>

```

En este ejemplo, hemos creado un contenedor que centra el contenido y establece un ancho máximo. La clase `mx-auto` se utiliza para centrar el contenedor horizontalmente.

Esta es una forma sencilla de crear un diseño limpio y centrado en la página. Puedes utilizar el contenedor para envolver cualquier contenido que desees centrar en la página.
