## Introducción

En esta sección, aprenderemos sobre los fundamentos de Tailwind CSS. Veremos cómo utilizar las clases utilitarias para crear diseños personalizados y cómo aplicar estilos a los elementos de una página web.

Para esto vamos seguir trabajando en nuestro proyecto de `mi-portafolio` y vamos a crear una página de inicio sencilla utilizando Tailwind CSS.

## Tipografía

La tipografía es un aspecto importante del diseño web. En Tailwind CSS, podemos utilizar clases utilitarias para aplicar estilos de tipografía a nuestros elementos. Algunas de las clases más comunes son:

**Fuente del texto:**: Es para camabiar la fuente del texto. En CSS se utiliza la propiedad `font-family`, en Tailwind CSS se utiliza la clase `font-{fuente}`. Por ejemplo, para utilizar la fuente `sans` podemos utilizar la clase `font-sans`.

```html
<p class="font-sans">Este es un texto con fuente sans-serif.</p>
```
```html
<p class="font-serif">Este es un texto con fuente serif.</p>
```
```html
<p class="font-mono">Este es un texto con fuente monoespaciada.</p>
```

**Tamaño del texto:**: Es para cambiar el tamaño del texto. En CSS se utiliza la propiedad `font-size`, en Tailwind CSS se utiliza la clase `text-{tamaño}`. Por ejemplo, para utilizar un tamaño de texto de `2xl` podemos utilizar la clase `text-2xl`. Estas clases de utilidad es desde `text-xs` hasta `text-9xl`. 

- `text-xs`: 0.75rem (12px)
- `text-9xl`: 5.625rem (90px)

```html
<h1 class="text-2xl">Hola Mundo</h1>

<p class="text-sm">Este es un texto pequeño.</p>
<p class="text-base">Este es un texto base.</p>
```

- **Estilo del texto:**: Es para cambiar el estilo del texto. En CSS se utiliza la propiedad `font-style`, en Tailwind CSS se utiliza la clase `italic` o `not-italic`. Por ejemplo, para utilizar un estilo de texto `italic` podemos utilizar la clase `italic`.

```html
<p class="italic">Este es un texto en cursiva.</p>
```

- **Negrita:**: Es para cambiar el estilo del texto. En CSS se utiliza la propiedad `font-weight`, en Tailwind CSS se utiliza la clase `font-bold` o `font-normal`. Por ejemplo, para utilizar un estilo de texto `bold` podemos utilizar la clase `font-bold`.

```html
<h1 class="text-2xl font-bold">Hola Mundo</h1>
<p class="font-normal">Este es un texto normal.</p>
```

- **Subrayado:**: Es para cambiar el estilo del texto. En CSS se utiliza la propiedad `text-decoration`, en Tailwind CSS se utiliza la clase `underline` o `no-underline`. Por ejemplo, para utilizar un estilo de texto `underline` podemos utilizar la clase `underline`.

```html
<p class="underline">Este es un texto subrayado.</p>
<p class="no-underline">Este es un texto sin subrayado.</p>
```

- **Transformación del texto:**: Es para cambiar el estilo del texto. En CSS se utiliza la propiedad `text-transform`, en Tailwind CSS se utiliza la clase `uppercase`, `lowercase` o `capitalize`. Por ejemplo, para utilizar un estilo de texto `uppercase` podemos utilizar la clase `uppercase`.

```html
<p class="uppercase">Este es un texto en mayúsculas.</p>
<p class="lowercase">Este es un texto en minúsculas.</p>
<p class="capitalize">Este es un texto con la primera letra en mayúscula.</p>
```

- **Aliniamiento del texto:**: Es para cambiar el estilo del texto. En CSS se utiliza la propiedad `text-align`, en Tailwind CSS se utiliza la clase `text-left`, `text-center` o `text-right`. Por ejemplo, para utilizar un estilo de texto `center` podemos utilizar la clase `text-center`.

```html
<p class="text-left">Este es un texto alineado a la izquierda.</p>
<p class="text-center">Este es un texto alineado al centro.</p>
<p class="text-right">Este es un texto alineado a la derecha.</p>
```

- **Espaciado entre letras:**: Es para cambiar el estilo del texto. En CSS se utiliza la propiedad `letter-spacing`, en Tailwind CSS se utiliza la clase `tracking-{espaciado}`. Por ejemplo, para utilizar un espaciado de letras de `tight` podemos utilizar la clase `tracking-tight`.

```html
<p class="tracking-tight">Este es un texto con espaciado de letras ajustado.</p>
<p class="tracking-normal">Este es un texto con espaciado de letras normal.</p>
<p class="tracking-wide">Este es un texto con espaciado de letras amplio.</p>
```

- **Contenido de After y Before:**: Es para cambiar el estilo del texto. En CSS se utiliza la propiedad `content`, en Tailwind CSS se utiliza la clase `before:content-{contenido}` o `after:content-{contenido}`. Por ejemplo, para utilizar un contenido de `Hello` podemos utilizar la clase `before:content-['Hello']`.

```html
<p>Higher resolution means more than just a better-quality image. With a
Retina 6K display, <a class="text-blue-600 after:content-['_↗']" href="...">
Pro Display XDR</a> gives you nearly 40 percent more screen real estate than
a 5K display.</p>
```

## Colores y fondo
Los colores son un aspecto importante del diseño web. En Tailwind CSS, podemos utilizar clases utilitarias para aplicar colores a nuestros elementos. Algunas de las clases más comunes son: 

Tailwind CSS tiene una paleta de colores predeterminada que podemos utilizar en nuestros proyectos. Esta paleta incluye colores como `red`, `green`, `blue`, `yellow`, `purple`, `pink`, entre otros. Cada color tiene diferentes tonalidades, que van desde `50` hasta `900`. Por ejemplo, para utilizar el color `red` con una tonalidad de `500`, podemos utilizar la clase `bg-red-500`.

```html
<div class="bg-red-500 text-white p-4">Este es un fondo rojo con texto blanco.</div>
```

```html
<div class="bg-green-500 text-white p-4">Este es un fondo verde con texto blanco.</div>
```

```html
<div class="bg-blue-500 text-white p-4">Este es un fondo azul con texto blanco.</div>
```

- **Color del texto:**: Es para cambiar el color del texto. En CSS se utiliza la propiedad `color`, en Tailwind CSS se utiliza la clase `text-{color}`. Por ejemplo, para utilizar un color de texto `red` podemos utilizar la clase `text-red-500`.

```html
<p class="text-red-500">Este es un texto rojo.</p>
<p class="text-green-500">Este es un texto verde.</p>
<p class="text-blue-500">Este es un texto azul.</p>
```

- **Color de fondo:**: Es para cambiar el color de fondo. En CSS se utiliza la propiedad `background-color`, en Tailwind CSS se utiliza la clase `bg-{color}`. Por ejemplo, para utilizar un color de fondo `red` podemos utilizar la clase `bg-red-500`.

```html
<div class="bg-red-500">Este es un fondo rojo.</div>
<div class="bg-green-500">Este es un fondo verde.</div>
<div class="bg-blue-500">Este es un fondo azul.</div>
```

- **Color de borde:**: Es para cambiar el color del borde. En CSS se utiliza la propiedad `border-color`, en Tailwind CSS se utiliza la clase `border-{color}`. Por ejemplo, para utilizar un color de borde `red` podemos utilizar la clase `border-red-500`.

```html
<div class="border-2 border-red-500">Este es un borde rojo.</div>
<div class="border-2 border-green-500">Este es un borde verde.</div>
<div class="border-2 border-blue-500">Este es un borde azul.</div>
```

- **Color de sombra:**: Es para cambiar el color de la sombra. En CSS se utiliza la propiedad `box-shadow`, en Tailwind CSS se utiliza la clase `shadow-{color}`. Por ejemplo, para utilizar un color de sombra `red` podemos utilizar la clase `shadow-red-500`.

```html
<div class="shadow-lg shadow-red-500">Este es un borde rojo.</div>
<div class="shadow-lg shadow-green-500">Este es un borde verde.</div>
<div class="shadow-lg shadow-blue-500">Este es un borde azul.</div>
```

- **Colore de SVG**: Es para cambiar el color de un SVG. En CSS se utiliza la propiedad `stroke`, en Tailwind CSS se utiliza la clase `stroke-{color}`. Por ejemplo, para utilizar un color de borde `red` podemos utilizar la clase `stroke-red-500`.

```html

<svg class="w-6 h-6 stroke-red-500" fill="none" viewBox="0 0 24 24" xmlns="http://www.w3.org/2000/svg">
    <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M12 4v16m8-8H4"></path>
</svg>
```

## Tamaños y contenedor

- **Alto:**: Es para cambiar el alto del elemento. En CSS se utiliza la propiedad `height`, en Tailwind CSS se utiliza la clase `h-{tamaño}`. Por ejemplo, para utilizar un alto de `1/2` podemos utilizar la clase `h-1/2`.

```html
<div class="h-1/2 bg-red-500">Este es un alto de 1/2.</div>

```

- **Ancho maximo:**: Es para cambiar el ancho máximo del elemento. En CSS se utiliza la propiedad `max-width`, en Tailwind CSS se utiliza la clase `max-w-{tamaño}`. Por ejemplo, para utilizar un ancho máximo de `1/2` podemos utilizar la clase `max-w-1/2`.

```html
<div class="max-w-1/2 bg-red-500">Este es un ancho máximo de 1/2.</div>
<div class="max-w-xs">Este es un ancho máximo de xs.</div>
```

- **Alto maximo:**: Es para cambiar el alto máximo del elemento. En CSS se utiliza la propiedad `max-height`, en Tailwind CSS se utiliza la clase `max-h-{tamaño}`. Por ejemplo, para utilizar un alto máximo de `1/2` podemos utilizar la clase `max-h-1/2`.

## Sistema de espaciado
El sistema de espaciado es un aspecto importante del diseño web. En Tailwind CSS, podemos utilizar clases utilitarias para aplicar espaciado a nuestros elementos. Algunas de las clases más comunes son:

- **Margen:**: Es para cambiar el margen del elemento. En CSS se utiliza la propiedad `margin`, en Tailwind CSS se utiliza la clase `m-{tamaño}`. Por ejemplo, para utilizar un margen de `4` podemos utilizar la clase `m-4`.

```html
<div class="m-4 bg-red-500">Este es un margen de 4.</div>
<div class="m-2 bg-red-500">Este es un margen de 2.</div>
<div class="m-8 bg-red-500">Este es un margen de 8.</div>
```

Margen superior, inferior, izquierdo y derecho:

```html
<div class="mt-4 bg-red-500">Este es un margen superior de 4.</div>
<div class="mb-4 bg-red-500">Este es un margen inferior de 4.</div>
<div class="ml-4 bg-red-500">Este es un margen izquierdo de 4.</div>
<div class="mr-4 bg-red-500">Este es un margen derecho de 4.</div>
```

Margen horizontal y vertical:

```html
<div class="mx-4 bg-red-500">Este es un margen horizontal de 4.</div>
<div class="my-4 bg-red-500">Este es un margen vertical de 4.</div>
```

- **Relleno:**: Es para cambiar el relleno del elemento. En CSS se utiliza la propiedad `padding`, en Tailwind CSS se utiliza la clase `p-{tamaño}`. Por ejemplo, para utilizar un relleno de `4` podemos utilizar la clase `p-4`.

```html
<div class="p-4 bg-red-500">Este es un relleno de 4.</div>
<div class="p-2 bg-red-500">Este es un relleno de 2.</div>
<div class="p-8 bg-red-500">Este es un relleno de 8.</div>
```

## Border y sombras

El borde y las sombras son un aspecto importante del diseño web. En Tailwind CSS, podemos utilizar clases utilitarias para aplicar bordes y sombras a nuestros elementos. Algunas de las clases más comunes son:
- **Borde:**: Es para cambiar el borde del elemento. En CSS se utiliza la propiedad `border`, en Tailwind CSS se utiliza la clase `border-{tamaño}`. Por ejemplo, para utilizar un borde de `2` podemos utilizar la clase `border-2`.

- **Sombras:**: Es para cambiar la sombra del elemento. En CSS se utiliza la propiedad `box-shadow`, en Tailwind CSS se utiliza la clase `shadow-{tamaño}`. Por ejemplo, para utilizar una sombra de `lg` podemos utilizar la clase `shadow-lg`.
